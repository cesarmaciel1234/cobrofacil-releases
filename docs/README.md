# Punto de Venta Pro 

Sistema completo y editable de Punto de Venta.

Características:
- Pesos argentinos ($)
- Productos por kilo y por unidad
- Pensado para carnicería
- Impresión térmica (estructura preparada)
- Guardado de productos y ventas
- Windows 11

Ejecución:
pip install -r requirements.txt
python main.py
