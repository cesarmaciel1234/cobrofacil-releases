# 📘 MANUAL OPERATIVO DEL CAJERO: ELITE 2026

Bienvenido al sistema Cobro Fácil POS. Esta guía le permitirá operar el terminal con la máxima eficiencia y seguridad.

---

## 🚀 1. ARRANQUE Y LOGIN
1.  Inicie el programa desde el icono del escritorio.
2.  Seleccione su perfil: **Cajero 1** o **Auxiliar 2** (esto configura automáticamente su impresora y cajón).
3.  Ingrese sus credenciales. El sistema se abrirá en modo **Pantalla Completa** (Kiosko) para evitar distracciones.

## 💰 2. APERTURA DE TURNO
Al iniciar, el sistema le pedirá el **Fondo de Caja**. Ingrese el monto físico que tiene en billetes. Esto quedará registrado para el arqueo final.

## 🛒 3. OPERACIÓN DEL TERMINAL (F1)
*   **Escaneo**: Simplemente pase el código de barras.
*   **Multiplicador**: Escriba la cantidad, un asterisco y luego el código (Ej: `5*779...`).
*   **Balanza**: Escanee el código generado por la balanza; el sistema calculará el peso automáticamente.
*   **Artículo Común**: Escriba `+` seguido del precio (Ej: `+150`) para artículos sin código.
*   **Navegación**: Use **F3** para ver el historial y **F12** para cobrar.

## 💳 4. COBRO Y FINALIZACIÓN (F12)
El flujo de cobro es 100% asistido por teclado para máxima velocidad:
*   **Selección de Método**: Use las **Flechas** para elegir (Efectivo, Tarjeta, etc.).
*   **Enter Inteligente (Triple Enter)**: 
    1.  **ENTER**: Selecciona el método y salta al casillero de monto.
    2.  **ENTER**: Confirma el monto ingresado (vuelto calculado).
    3.  **ENTER**: Cierra la venta **SIN imprimir ticket** (igual que F2).
*   **Modificadores**: Presione **F3** para aplicar un Descuento o **F4** para un Recargo antes de cobrar.
*   **Asistencia Mixta**: Si ingresa un monto menor al total, el sistema activará automáticamente el modo **PAGO MIXTO** y le pedirá el saldo faltante en el segundo casillero.
*   **Cierre Directo**: 
    *   **F1**: Finaliza e imprime ticket.
    *   **F2**: Finaliza **SIN imprimir ticket** (ahorro de papel).

## 👮‍♂️ 5. ASISTENCIA DE SUPERVISOR (F11)
Si necesita cambiar un precio, corregir un error o reparar la impresora, llame al supervisor. Él presionará **F11**, ingresará su PIN y podrá realizar los ajustes necesarios sin cerrar su venta actual. Una vez terminado, él tocará la **Burbuja Parpadeante** y el terminal se refrescará con los nuevos datos.

## 🏁 6. CIERRE CIEGO (F4)
Al terminar su turno:
1.  Presione **F4**.
2.  Realice el **Arqueo Ciego**: cuente el dinero físico y anótelo. El sistema no le dirá cuánto "debería" haber hasta que usted guarde el cierre.
3.  El sistema se cerrará y volverá a la pantalla de Login automáticamente.
