# 📘 MANUAL DE OPERACIÓN ELITE 2026: MODO CAJERO

Bienvenido al ecosistema **Cobro Fácil POS**. Este manual detalla el flujo de trabajo industrial optimizado para cajeros, desde el encendido hasta el cierre de jornada.

---

## 🚀 PASO 1: ARRANQUE Y SELECCIÓN DE PERFIL
Al hacer clic en el icono de **PunPro POS**, el sistema inicia en **Modo Kiosko** (Pantalla Completa).

1.  **Selección de Estación**: Se presentará la pantalla de bloqueo con dos opciones:
    *   **CAJERO 1 (Azul)**: Estación principal.
    *   **AUXILIAR 2 (Verde)**: Estación de apoyo o segunda caja.
2.  **Identificación**: Ingresa tu **PIN de 4 dígitos**. 
    *   *Nota*: El sistema vincula automáticamente la impresora y el cajón de dinero correspondiente a tu selección.

---

## 🏁 PASO 2: APERTURA DE TURNO
Antes de vender, debes registrar el capital inicial:
1.  Ingresa el **Monto de Apertura** (Fondo para cambio).
2.  Presiona **ENTER** para confirmar. El cajón se abrirá brevemente para que valides el efectivo.
3.  **¡Terminal Lista!**: El cursor se posicionará automáticamente en la barra de escaneo (F1).

---

## 🛒 PASO 3: OPERACIÓN DEL TERMINAL (F1)
El terminal está diseñado para usarse **sin ratón**.

### ⌨️ Comandos Esenciales
*   **Escaneo Directo**: Pasa el producto por el láser. Se agregará instantáneamente.
*   **Multiplicador**: Para llevar varios del mismo, escribe `CANTIDAD * CÓDIGO` (Ej: `6 * 779...`).
*   **Balanza Automática**: Si el producto es de pesaje, el sistema extraerá el peso del código de barras (Systel/Kretz) automáticamente.
*   **Artículo Común (+)**: Para cobrar algo sin código, escribe `+ PRECIO` (Ej: `+500`) y presiona ENTER.
*   **Modificación de Cantidad**: Usa las flechas ↑ ↓ para elegir un producto y ← → para sumar o restar unidades. Presiona **ENTER** para edición precisa.

---

## 💳 PASO 4: COBRO Y FINALIZACIÓN (F12)
Cuando el cliente esté listo, presiona **F12** o **ENTER** con el buscador vacío.

1.  **Medios de Pago**: Selecciona Efectivo, Tarjeta o Mixto.
2.  **Cálculo de Cambio**: El sistema te indicará el vuelto exacto en letras grandes.
3.  **Finalizar (F1)**: Presiona F1 para cobrar e imprimir ticket. El cajón se abrirá automáticamente.
    *   *Importante*: Si el cajón queda abierto, el sistema se bloqueará por seguridad. **Ciérralo para continuar.**

---

## 👮‍♂️ PASO 5: ASISTENCIA Y SEGURIDAD (F11)
Si tienes un problema técnico o necesitas cambiar un precio:
1.  Presiona **F11**. Llama al **Supervisor**.
2.  El supervisor ingresará su PIN. Esto habilitará el **Modo Intervención**.
3.  **Burbuja Bombero**: Aparecerá un icono parpadeante (Rojo/Azul). Mientras el supervisor arregla el problema en el panel administrativo, tu venta se mantiene segura.
4.  **Regreso Seguro**: Al tocar la burbuja, volverás al terminal con los **precios y configuraciones actualizadas** al instante.

---

## 🏁 PASO 6: CIERRE CIEGO DE JORNADA (F4)
Al finalizar tu turno, presiona **F4** para rendir cuentas.

1.  **Arqueo Ciego**: Cuenta el efectivo físico en tu cajón e ingresa el monto total.
2.  **Diferencias**: El sistema comparará tu conteo con las ventas registradas. Si hay faltantes, el borde de la pantalla parpadeará en rojo (Alerta SOS).
3.  **Restauración**: Al confirmar el cierre, el ticket de arqueo se imprimirá y el sistema **se cerrará automáticamente**, volviendo a la pantalla de Login para el siguiente compañero.

---
> [!IMPORTANT]
> **VIGILANCIA ACTIVA**: El sistema detecta si el cajón se abre con llave física. Cualquier apertura no autorizada activará una alarma roja y quedará grabada en el reporte del administrador.
